# Survey Solutions Style Guide

This repository contains the source code for the Survey Solutions Style Guide

## Installation

To build this project, one needs:

- Quarto

To "install" this project, simply [download the source code](https://docs.github.com/en/repositories/working-with-files/using-files/downloading-source-code-archives#downloading-source-code-archives-from-the-repository-view) or `git clone` the repo.

## Usage

To build the book:

- Navigate to the project directory
- Run `quarto preview` in the terminal
